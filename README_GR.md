<h1>❔ Πώς λειτουργεί</h1>

###

<h2>HTTP</h2>

Δεδομένου ότι οι περισσότερες ιστοσελίδες στον κόσμο υποστηρίζουν τώρα HTTPS, το MacDPI δεν παρακάμπτει την Εμβαθυσμένη Επιθεώρηση Πακέτων για αιτήματα HTTP. Ωστόσο, παρέχει ακόμα σύνδεση proxy για όλα τα αιτήματα HTTP.

<h2>HTTPS</h2>

Αν και το TLS κρυπτογραφεί κάθε διαδικασία χειραψίας, τα ονόματα τομέων εξακολουθούν να εμφανίζονται σε απλό κείμενο στο πακέτο Client Hello. Με άλλα λόγια, αν κάποιος κοιτάξει το πακέτο, μπορεί εύκολα να μαντέψει πού πηγαίνει το πακέτο. Το όνομα τομέα μπορεί να προσφέρει σημαντικές πληροφορίες κατά τη διάρκεια της επεξεργασίας DPI, και μπορούμε να δούμε ότι η σύνδεση αποκλείεται αμέσως μετά την αποστολή του πακέτου Client Hello. Είχα δοκιμάσει διάφορους τρόπους για να παρακάμψω αυτό και διαπίστωσα ότι φαίνεται πως μόνο το πρώτο τμήμα ελέγχεται όταν στέλνουμε το πακέτο Client Hello χωρισμένο σε τμήματα. Αυτό που κάνει το MacDPI για να παρακάμψει αυτό το πρόβλημα είναι να στείλει το πρώτο byte ενός αιτήματος στον διακομιστή και στη συνέχεια να στείλει το υπόλοιπο.
